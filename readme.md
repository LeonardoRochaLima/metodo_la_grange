## Projeto Lagrange
Um codigo para calcular Lagrange e Newton.

## Instalação
Baixar todos os arquivos.

Para calcular Lagrange :

Abrir o arquivo LAGRANGE.html.

Para calcular Newton :

Abrir o arquivo NEWTON.html.
